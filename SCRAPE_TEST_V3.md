## 三、关键洞察

### 1. 5 时段分布规律

```
6 条线中：
- 6-9 时段（早班）：5 个航班，占 19%
- 9-15 时段（白天）：1 个航班，占 4%
- 15-21 时段（下午）：7 个航班，占 27%
- 21-24 时段（晚班）：13 个航班，占 50%
- 0-6 时段（红眼）：0 个航班，占 0%
```

**晚班（21-24）占了一半** —— 这跟你的代理经验一致，晚班价格便宜、客流相对少。MVP 应该把"晚班推荐"作为重点输出。

### 3. 时段 × 折扣交叉分析

```
早班 (6-9)：折扣率 2.1-3.4 折 → 全价暴露比较保守
晚班 (21-24)：折扣率 1.7-2.5 折 → 全价暴露更激进
```

**晚班折扣率更低** = 晚班用更激进的折扣吸引乘客。MVP 可以推荐："想最便宜？选晚班+提前 X 天购买"。

### 4. 航司定价策略差异

- **东航**（MU）：覆盖所有时段，折扣率中等（2-2.5 折）
- **国航**（CA）：覆盖 9-15 时段，折扣率较低（2.9 折）
- **吉祥**（HO）：覆盖早班和晚班，折扣率激进（2.3-2.8 折）
- **上海航空**（FM）：覆盖早班

**MVP 可以做"航司推荐"**：根据用户偏好（时段/价格）推荐最优航司。
